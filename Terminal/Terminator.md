# Terminator Cheatsheet 

## **Fenêtres & Sessions**

| Raccourci          | Action                                      |
| ------------------ | ------------------------------------------- |
| `Ctrl + Shift + O` | **Diviser horizontalement**                 |
| `Ctrl + Shift + E` | **Diviser verticalement**                   |
| `Ctrl + Shift + W` | **Fermer le terminal actif**                |
| `Ctrl + Shift + Q` | **Quitter Terminator** (tous les terminaux) |
| `Ctrl + Shift + N` | **Fenêtre suivante**                        |
| `Ctrl + Shift + N` | **Fenêtre précédente**                      |

## **Navigation entre panneaux**

| Raccourci                     | Action                                               |
| ----------------------------- | ---------------------------------------------------- |
| `Alt + Flèche directionnelle` | **Se déplacer entre les terminaux**                  |
| `Ctrl + Tab`                  | **Naviguer entre les terminaux ouverts** (par focus) |

## **Tiling & Redimensionnement**

| Raccourci                | Action                          |
| ------------------------ | ------------------------------- |
| `Ctrl + Shift + flèches` | **Redimensionner les panneaux** |


## **Onglets**

| Raccourci                       | Action                        |
| ------------------------------- | ----------------------------- |
| `Ctrl + Shift + T`              | **Nouveau terminal (onglet)** |
| `Ctrl + Page Up / Page Down`    | **Changer d’onglet**          |
| `Ctrl + Shift + Page Up / Down` | **Réorganiser les onglets**   |

## **Divers**

| Raccourci               | Action                                            |
| ----------------------- | ------------------------------------------------- |
| `Ctrl + Shift + X`      | **Basculer le mode plein écran du panneau actif** |
| `Ctrl + Shift + F`      | **Recherche dans le terminal actif**              |
| `Ctrl + +` / `Ctrl + -` | **Augmenter / diminuer la taille de la police**   |


## Personnalisation

Terminator est **configurable** via le fichier :

```bash
~/.config/terminator/config
```
